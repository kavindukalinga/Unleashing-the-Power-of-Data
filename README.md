## 1. Basic Analysis

1.  Data reading 

2.  Data parsing 

3.  Creation of an efficient data structure 

4.  Organization 

5.  Enrichment of data 

6.  Basic statistical analysis 
     
Use redex to data parsing.

Use Pandas.Dataframe to store data.
